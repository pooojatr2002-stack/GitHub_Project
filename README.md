# OLA 
## Project Overview
A data analysis project on OLA booking Using SQL and Power BI to explore trends in revenue, cancellations, and customer ratings.

**Tools & Technologies:** 
- Power BI - Dashboard visualization and interactive analysis.
- SQL Server - Data Extration
- Excel/CSV - Used as data source.
  
**Highlights:**
- Built an interavtive Power BI Dashboard showing bokings, revenue, and cancellation.
- Found Prime SUV and Prime Plus as top revenue generators, with cash and UPI as the most used payment modes
- Identified peak-hour demand an driver-related cancellation as key operational challenges.

and jobs satisfaction to support HR decision-making.

**Tools & Technologies:**
- Power BI
- Power Query - Cleaning and Analysing
- Power BI Desktop - Visualization

- Identifies employees due for promotion or retrenchments by department and service years.
